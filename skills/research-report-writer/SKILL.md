# Research Report Writer

This legacy alias exists for compatibility. For paper drafting, manuscript revision, reviewer-style iteration, or session-memory-to-paper workflows, use `research-paper-writer` instead.

When this skill is invoked:

1. Invoke or read the canonical `research-paper-writer` skill from the active skill list or global skills directory.
2. Follow the paper-writing workflow there, including its mandatory real multi-agent startup gate.
3. If the request is truly a non-paper project report, reuse only the evidence-ledger, integrity-audit, and export parts that still apply.
